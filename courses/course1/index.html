Software needed to get started

R Software

  • R is a statistical software that can be used to provide meaniful informaion (computing and graphics) about dataset in any format.
  • It is freely available online and it can run on any platform (dont be worried if you use a mac or unix platform).
  • R is always available in different version, always use the latest version.
  • R can be downloaded from the Comprehensive R Archive Network.
    • Click the download R on the second line and choose a CRAN mirror to download.
    • It is recommended to choose the CRAN mirror close to your country.
  • Some functionality in R are organized in packages which contain several functions.
  • mean() is a function in R that calulate the mean of several observations.
  • n() is a function in the dplyr package that gives the number of observations in a group and the format of represenation in R is dplyr::n().

R Studio

  • R studio is is an open source software for R.
  • It is always recommended to use R studio in other to fully understand and use all its functionality using the R syntax.
  • R studio can be used for various purposes as we will see over time.
  • R studio provide a good platform for the use of of R software through it user interface.
  • The latest version of R studio for different platform can be downloaded Here.

R Markdown

  • R markdown is one tte cool features of R studio that enables your project to be reproduceable.
  • R markdown can be used to generate your report in any format.
  • A single R markdown file nicely combines your code for analysis and the some texts to generate your report.
  • R markdown is free and open source and it is apackage in R.
  • When you open an R session, R markdown can be installed using install.packages("rmarkdown").

Git/Github

  • Git is an open-source version control system.
  • Version control systems manage the revisions made on your project or documents in a central repository.
  • Git is the mostly used version control because it stores project or file changes efficiently.
  • Git is a command line tool central around the github.
  • Github is a place to store your projects and track the changes made to the project.
  • Each project files are stored in a repository.
  • Register for an account on GitHub.
  • Download Git here.
  • Open Git bash on your PC and run the following commands.
    • git config --global user.name "your_GitHub_Username_here"
    • git config --global user.mail "your_email_here"
